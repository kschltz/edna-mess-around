To build this project, you'll need the Clojure CLI tool:

https://clojure.org/guides/deps_and_cli


To develop in a browser with live code reloading:

`clj dev.clj`


To build a release version for the web:

`clj prod.clj`


To play the song once:

`clj dev.clj play`


To build an mp3:

`clj prod.clj mp3`
